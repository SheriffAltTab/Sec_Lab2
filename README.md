# Лабораторна робота №2 (продовження ЛР1)

Веб-застосунок реалізує ідентифікацію та аутентифікацію користувачів з розмежуванням повноважень і веденням журналів.

## Запуск

1. Встановити залежності:
   - `pip install -r requirements.txt`
2. Запустити застосунок:
   - `python app.py`
3. Відкрити у браузері:
   - `http://127.0.0.1:5000`

## Реалізовано для ЛР2

- 12 користувачів у реєстраційному журналі (включно з `ADMIN`).
- 3 рівні доступу (`1..3`) та 6 прикладних функцій із перевіркою прав.
- Хешування пароля (`SHA-256`) + перевірка відображення `a*sin(1/x)` під час входу.
- Періодична аутентифікація: кожні 2 хвилини, 4 питання за ітерацію з набору 15.
- Реєстраційний журнал (`registration_log.json`) для входу/виходу та дій адміністратора.
- Операційний журнал (`operation_log.json`) для дій користувачів у системі.

## Варіант №28

Парольний шаблон: чергування `літера -> знак пунктуації -> цифра`, довжина кратна 3.

Приклад коректного пароля: `A!1b?2`.
